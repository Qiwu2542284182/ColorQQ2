按钮属性说明

背景图标名称

图标的文件名，所有按钮图标都必须保存在 内部储存/QQColor2/fab 里才能生效

例如： 文件为:内部储存/QQColor2/fab/ic_exit.png，属性值应填ic_exit.png

按钮大小

0为大（与菜单按钮一样大）；1为小

点击事件

固定事件:

名称 功能
qzone QQ空间
wallet QQ钱包
fav 我的收藏
photo 我的相册
qlink 面对面快传
search 搜索
money 收付款
message 消息
contact 联系人
readInjoy 看点
leba 动态
exit 退出QQ

自定义事件：

名称 说明
jump_ + 活动名 跳转到指定活动
url_ + 网址 跳转到指定网址
friend_ + 好友QQ号 跳转到指定QQ好友聊天界面
app_ + 应用包名 打开指定应用

自定义事件举例:

举例 说明
jump_com.tencent.
mobileqq.activity.
QQSettingSettingActivity		 跳转到QQ设置页面
url_http://www.baidu.com 打开百度
friend_123456 打开与QQ号为123456
的聊天界面
app_me.qiwu.colorqq2 打开ColorQQ模块
