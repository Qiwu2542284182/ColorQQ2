主题zip结构

  • 主题包zip

    • assets(assets里的图片，主要为启动图)

    • color(颜色)

    • drawable-xhdpi(图标)

    • layout(布局)

    • theme.xml(主题信息，必须存在)

theme.xml主要内容(Json格式)

{

 "id":"2",  //主题id，不能为0
 "author":"祈无",    //作者名字
 "name":"谷歌蓝 ",    //主题名字
 "version":"3",    //主题版本(用户)
 "versionCode":"4",    //主题版本(模块)
 "QQVersion":"8.2.0,8.1.5",   //适配QQ版本，如有多个用,分割
 "description":"修改整体配色为谷歌蓝"    //主题描述

}
